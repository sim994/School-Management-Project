📘 # School Management System (MERN Stack)

This project is a School Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) to understand core full-stack development concepts.

It focuses on implementing real-world functionalities and how frontend, backend, and database work together.

🔹 Key Points:

Built with MERN Stack for full-stack development
Perform CRUD operations (Add, Update, Delete, View students)
RESTful APIs for communication between client and server
React.js for dynamic and responsive UI
Node.js & Express.js for backend logic and routing
MongoDB for storing and managing data
Basic user roles and authentication flow
Efficient state and data handling

This project helped me understand how to build scalable applications, manage data flow, and connect all parts of the MERN stack into a single working system. 🚀
